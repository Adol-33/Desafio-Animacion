## Descripción
Sitio web de tatuador que muestra info y imagenes, diseño responsivo que utiliza CSS Grid y Flexbox para organizar los elementos principales. Incluye Media Query, fue diseñado Mobile First. Se agregan Animaciones en imagenes y texto.

---

## Funcionalidades
* Diseño principal con Grid.
* Imagenes organizadas con Flexbox.
* Menú para Mobile y Desktop.
